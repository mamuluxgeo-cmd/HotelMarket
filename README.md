## Apps Script-ის ჩასმა

1. გახსენი Google Sheet
2. შედი `Extensions -> Apps Script`
3. ჩასვი ფაილი `apps-script/Code.gs`
4. ჩასვი/დაამატე ფაილი `apps-script/RoomDebtPatch.gs` — ეს არის ოთახების დავალიანებების და სავალდებულო გადახდის ტიპის დამატება
5. Project Settings-ში ჩართე `Show appsscript.json manifest file in editor`
6. ჩასვი `apps-script/appsscript.json`
7. გაუშვი ფუნქცია `setupDatabase`
8. მიეცი ავტორიზაცია
9. გააკეთე Deploy: `Deploy -> New deployment -> Web app`

რეკომენდირებული Web App პარამეტრები:

- Execute as: `Me`
- Who has access: `Anyone with the link`
